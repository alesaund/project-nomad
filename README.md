# Project Nomad

🚐 **Project Nomad** è il progetto di progettazione e realizzazione di un camper full-time off-grid intelligente.

## Obiettivi
- Impianto elettrico 24 V
- Ecosistema Victron
- Home Assistant
- Networking avanzato
- Sicurezza
- Documentazione compatibile con Obsidian
