# 01 - Acqua

## Filosofia
Project Nomad adotterà un impianto idrico di tipo navale, modulare, ridondante e monitorato. L'obiettivo è privilegiare semplicità, affidabilità e facilità di manutenzione.

## Principi
- Home Assistant monitora l'impianto ma non è necessario al suo funzionamento.
- Ogni componente deve poter essere sostituito facilmente.
- Soluzioni semplici e collaudate hanno priorità rispetto a sistemi complessi.

## Filtrazione approvata
Carico acqua → Prefiltro lavabile → Serbatoio → Pompa autoclave → Filtro combinato (sedimenti + carbone attivo) → Lampada UV → Collettore → Rubinetti.

## Evoluzione futura
Valutare un eventuale circuito di ricircolo con UV come upgrade opzionale, non previsto nella configurazione iniziale.

## Componenti da definire
- Prefiltro
- Pompa
- Filtro combinato
- Lampada UV
- Collettore
- Sensori
